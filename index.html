아래 버튼을 누르면 현재 종목 데이터가 holdings.json 파일로 다운로드됩니다. GitHub 저장소 페이지에 이 파일을 끌어다 놓으면(드래그 앤 드롭) 기존 파일이 자동으로 교체되며, 별도 토큰이나 로그인 절차 없이 그대로 커밋됩니다.
순서: 다운로드 → GitHub 저장소 페이지 열기 → 파일 목록 위에 다운로드한 파일을 드래그 → 상단에 나오는 "Commit changes" 버튼 클릭.
